# karaoke-karjalakse
Ihastuta sinun kuulijua karjalan kielel. Täs on karjalankielizen iskumeloudiin uuzi tulendu, muailman ezmäine karjalankieline karaoketallendeh. Keral karjalankielisty folkkii, rokkii dai kiännösiskelmii monil eri vuozikymmenil. Eigo haittua ku et jo maltanne kieldy, karaoke panou sanat sinun suuh. Tallendehes on pajuo kaikil karjalan kielen murdehi
